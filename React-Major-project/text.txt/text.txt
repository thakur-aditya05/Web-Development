Direct geocoding
Direct geocoding allows to get geographical coordinates (lat, lon) by using name of the location (city name or area name).
 If you use the limit parameter in the API call, you can cap how many locations with the same name will be seen in the API
response (for instance, London in the UK and London in the US)
